# CreatorPlatform.xyz

Learn to build a platform with embedded finance to enable creators to sell
their digital downloads to end customers. Uses Ruby on Rails 7, TailwindUI, AWS
S3, Heroku, Postgres, Sidekiq, Stimulus.js, and Stripe.
